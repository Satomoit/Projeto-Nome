# Projeto Nome - OpenGL  
Este projeto desenha o nome "Josiane" usando OpenGL e permite transformações como escala, translação e rotação.  

## 🚀 Funcionalidades  
- `+ / -` → Aumenta/Diminui o tamanho  
- `W, A, S, D` → Move para cima, baixo, esquerda e direita  
- `Q / E` → Rotaciona  

## 🛠️ Tecnologias  
- OpenGL  
- C++
